# bitcoin-stock-price-prediction
A deep learning project that predicts Bitcoin (BTC-USD) closing prices using an LSTM neural network built with TensorFlow/Keras.

---

##📌 Project Overview

- This project:
- Downloads historical Bitcoin data using Yahoo Finance

- Applies data preprocessing and normalization

- Builds a multi-layer LSTM neural network

- Trains on 80% of the dataset

- Predicts prices on test data

- Visualizes predicted vs actual prices

---

##🧠Model Architecture

- 4 LSTM layers (50, 60, 80, 120 units)

- Dropout regularization (0.2 – 0.5)

- Dense output layer

- Adam optimizer

- Mean Squared Error loss
